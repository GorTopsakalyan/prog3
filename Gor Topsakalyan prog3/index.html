# Գույն Նկարագրություն Ծնվել է
1 խոտ, որը բազմանում է N ցիկլը մեկ,մահանում է երբ նրան ուտում է խոտակերը, կամ այրում է կրակը, կամ նրան կոխկրտում է գիշատիչը։
2 խոտակեր, որը բազմանում է N հատ խոտ ուտելուց հետո և մահանում է N հատ չուտելուց հետո, կամ երբ նրան ուտում է գիշատիչը, կամ երբ նրան այրում է կրակը:
3 գիշատիչ, որը բազմանում է N հատ խոտակեր կամ մարդ ուտելուց հետո և մահանում է N հատ չուտելուց հետո, կամ երբ N անգամ չի ուտում խոտակեր։
4 մարդ,չի բազմանում,մահանում է N ժամանակ հետո,ուտում է խոտակերին,իր հետևից առաջացնում է խոտ
5 ժայռ,չի բազմանում, չի մահանում,ամեն N վայրկյանը մեկ "քանդվում" է և ոչնչացնում իր կողքը գտնվող վանդակները
